# Dashboard de Vendas – Xbox Game Subscriptions (Excel)

## Sobre o Projeto
Este projeto consiste na criação de um **dashboard de vendas no Excel**, voltado para a análise de **assinaturas de jogos do ecossistema Xbox**.  
O painel foi desenvolvido com foco na **organização, visualização e interpretação de dados**, transformando informações brutas em indicadores visuais claros para apoiar a **tomada de decisão baseada em dados**.

O dashboard apresenta uma visão consolidada das vendas de assinaturas ao longo do período analisado, utilizando gráficos, KPIs e filtros interativos.

---

## Objetivo do Dashboard
- Organizar dados de vendas de assinaturas de jogos
- Apresentar indicadores-chave de desempenho (KPIs)
- Facilitar a análise visual por período
- Comparar o desempenho entre diferentes tipos de assinatura
- Apoiar decisões estratégicas a partir dos dados apresentados

---

## Período de Análise
- **Período de cálculo:** 01/01/2024 a 31/12/2024  
- **Última atualização:** 08/01/2025

---

## Indicadores e Visualizações
O dashboard apresenta os seguintes indicadores e recursos visuais:

- **Total de assinaturas EA Play Season Pass**
- **Total de assinaturas Minecraft Season Pass**
- **Total de assinaturas Xbox Game Pass**
- Comparação entre clientes que possuem ou não assinatura
- Gráficos de barras horizontais
- Cards com valores totais em reais (R$)
- Layout visual padronizado para melhor leitura

---

## Filtros Disponíveis
O painel conta com filtros interativos que permitem a análise por tipo de período:
- Anual
- Mensal
- Trimestral

Esses filtros facilitam a visualização dinâmica dos dados conforme a necessidade do usuário.

---

## Como Reproduzir o Projeto
1. Faça o download do arquivo Excel disponível neste repositório
2. Abra o arquivo no Microsoft Excel ou software compatível
3. Acesse a aba do dashboard
4. Utilize os filtros laterais para explorar os dados
5. Analise os indicadores e gráficos apresentados
